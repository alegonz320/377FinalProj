How to Use the Map

The filter dropdown menu allows you to filter the data by "type of crime" and "year occurred". Click/press on thee provided filter and the data in the map will change to reflect the choice.

Pinch the screen while using a touch screen or use the scroll function on a mouse to zoom in on the map.

Click/press on an area in the map to get more detailed statistics related to the crime density of a specific region.
